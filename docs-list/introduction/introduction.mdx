# Introduction

- Target for explaining 3D graphic from high-level concept (SAM.js) to low-level concept (WebGPU)

## Why not WebGL, but WebGPU?

- There are various great explainers for WebGL or OpenGL, more than WebGPU.
- WebGPU follows modern design concept like DirectX, Vulkan, and Metal.
- It's better to practice and implement various techniques via WebGPU.
